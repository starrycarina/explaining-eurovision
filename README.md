geospatial_data_clean is too big to upload but can be built with the uploaded files and R script. It contains geospatial country border data and includes border changes that happen within the Eurovision time period, as well as capital city coordinates, area in km squared and number of years since the borders were last changed. This dataset is then used to create voting_data_geo which has all the Eurovision voting pairings with their border distance and distance between capital cities as km squared.
